How to Set Up Google Sheets Integration

  1. Create a Google Sheet with columns: Name, Class
  2. Go to File > Share > Publish to web
  3. Select the sheet and choose "Comma-separated values (.csv)" format
  4. Copy the published URL and paste it below
  5. For writing data, you'll need to set up a Google Apps Script (optional)
